Privacy Policy

Effective Date: April 2026. Your privacy and server security are our core mission.

  • 1. Data Collection CyberShield collects minimal data to ensure server security. This includes User & Server IDs, Audit Logs, and Invite Metadata.
  • 2. Data Usage Collected data is used strictly for moderation purposes. We do not monitor private conversations or track users across servers.
  • 3. Data Sharing We never sell, trade, or share your data with third parties. Your server's internal structure remains private.
  • 4. Data Retention Moderation logs are stored for a maximum of 30 days. Removing the bot flags all configuration data for deletion.
  • 5. User Rights Users can request to view or delete their stored data by contacting the development team.
  • 6. Policy Updates We may update this policy to reflect new security features. Continued use constitutes acceptance.
